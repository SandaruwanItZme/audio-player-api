# 🎵 Audio Player API

A professional, embeddable audio player for your website. Works like an API - just add the code and it works!

## 🚀 Quick Start

### Method 1: Embed in your website
Add these lines to your HTML:

```html
<!-- Add Font Awesome -->
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css">

<!-- Add Player CSS -->
<link rel="stylesheet" href="https://yourusername.github.io/audio-player-api/player.css">

<!-- Add Player HTML -->
<div class="audio-player-api" id="audioPlayerAPI" 
     data-playlist="https://yourusername.github.io/audio-player-api/api/playlist.json"></div>

<!-- Add Player JS -->
<script src="https://yourusername.github.io/audio-player-api/player.js"></script>
